🛡️管理员面板按钮功能说明
=================
⚙️在当前用户对应的子话题中发送 /admin，可打开管理员面板并使用以下功能：

--- 🔘按钮功能 ---
➡️拉黑用户 / 解除拉黑：根据当前用户状态动态显示，用于拉黑或解除拉黑当前用户
➡️开启验证码 / 关闭验证码：全局开关所有用户的验证码功能
➡️查询黑名单：列出当前所有被拉黑的用户 ID
➡️开启用户Raw / 关闭用户Raw：全局开关用户端欢迎内容中的 Raw 链接
➡️GitHub项目：跳转至 GitHub 项目地址
➡️重置用户：清空当前用户状态和消息频率，但保留当前子话题继续复用
➡️删除用户并删除话题：清空当前用户状态，并删除当前子话题；用户下次发消息时会重新创建新话题
